# R4C by Kn33l

# Game Description
This is my first game that I tried making using Unity. I built it so I can understand how Unity works and its basics. It is a simple game that is similiar to a flappy bird but instead of a pipes and a bird, it's a race car and cones as obstacles. 

This game can only run on PCs

# Controls
It has simple controls with tapping the space bar to move up to dodge the cones.
